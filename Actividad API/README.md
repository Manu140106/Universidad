# Actividad API Harry Potter

Proyecto sencillo con una página web y una API local hecha con Node.js nativo.

## Ejecutar

```bash
npm start
```

Abre luego:

- Web: http://localhost:3000
- API: http://localhost:3000/api
- Personajes: http://localhost:3000/api/characters

## Estructura

- `server.js`: servidor HTTP y rutas de la API
- `public/`: página web
- `data/harry-potter.json`: datos de ejemplo
